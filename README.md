# PredictMaint AI

PredictMaint AI is my BCA AI & ML project developed to predict industrial machine health using Machine Learning.

The application allows users to upload machine sensor data in CSV format and provides maintenance predictions along with analytics and reports.

## Features

- User Login
- Dashboard
- Machine Health Prediction
- Upload CSV Files
- Analytics Dashboard
- Maintenance History
- Notifications
- Reports
- Calendar
- Factory Management

## Technologies

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS
- JavaScript

## How to Run

Clone the repository

```
pip install -r requirements.txt
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
